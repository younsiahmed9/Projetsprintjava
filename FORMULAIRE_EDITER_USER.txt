2. MÉTHODE initialize() MODIFIÉE:
   AJOUT du gestionnaire de double-clic:
   `java
   if (usersTable != null) {
       usersTable.setRowFactory(tv -> {
           TableRow<User> row = new TableRow<>();
           row.setOnMouseClicked(event -> {
               if (event.getClickCount() == 2 && !row.isEmpty()) {
                   onEditUser(row.getItem());
               }
           });
           return row;
       });
   }
   `

FLUX D'UTILISATION:
===================
SCÉNARIO 1: Modifier un utilisateur
------------------------------------
1. Dans le dashboard admin
2. Double-cliquez sur une ligne du tableau
3. Le dialogue s'ouvre avec les données
4. Modifiez les champs souhaités
5. Cliquez OK
6. L'utilisateur est mis à jour
7. La table se rafraîchit
8. Message "Utilisateur modifié avec succès !"
SCÉNARIO 2: Changer le mot de passe
------------------------------------
1. Double-cliquez sur l'utilisateur
2. Entrez un nouveau mot de passe
3. Cliquez OK
4. Le mot de passe est hashé (SHA-256)
5. Mis à jour en base de données
6. Message de succès
SCÉNARIO 3: Laisser le mot de passe inchangé
---------------------------------------------
1. Double-cliquez sur l'utilisateur
2. Laissez le champ mot de passe vide
3. Modifiez d'autres champs si nécessaire
4. Cliquez OK
5. L'ancien mot de passe est conservé
6. Autres champs mis à jour
SCÉNARIO 4: Supprimer un utilisateur
-------------------------------------
1. Double-cliquez sur l'utilisateur
2. Cliquez sur "Supprimer" (à gauche)
3. Dialogue de confirmation s'affiche
4. Confirmez avec OK
5. L'utilisateur est supprimé
6. La table se rafraîchit
7. Message "Utilisateur supprimé avec succès !"
SCÉNARIO 5: Changer le rôle
----------------------------
1. Double-cliquez sur l'utilisateur
2. Changez le rôle (ADMIN ↔ CLIENT)
3. Cliquez OK
4. Le rôle est mis à jour
5. Message de succès
SCÉNARIO 6: Désactiver un utilisateur
--------------------------------------
1. Double-cliquez sur l'utilisateur
2. Décochez "Actif"
3. Cliquez OK
4. L'utilisateur est désactivé
5. Il ne pourra plus se connecter
