## 🚀 Utilisation Rapide

### Ajouter un bouton supprime (ROUGE)
```java
Button delete = new Button("🗑️");
delete.getStyleClass().add("btn-delete");
delete.setPrefSize(40, 40);
```

### Ajouter un bouton modifier (ORANGE)
```java
Button edit = new Button("✎️");
edit.getStyleClass().add("btn-modify");
edit.setPrefSize(40, 40);
```

### Créer une carte de budget
```java
card.getStyleClass().add("account-card");
```
