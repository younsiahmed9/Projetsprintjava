🚀 QUICK START

  1. Lancer l'application
     → mvn javafx:run

  2. Cliquer sur "🏷️ Catégories" dans le menu

  3. Ajouter une catégorie
     - Nom : "Important"
     - Description : "Documents importants"
     - Cliquer "✓ Ajouter"
     - ✅ Succès !

  4. Modifier une catégorie
     - Cliquer sur "Important" dans la liste
     - Changer le nom
     - Cliquer "✎ Modifier"
     - ✅ Succès !

  5. Supprimer une catégorie
     - Cliquer sur "Important"
     - Cliquer "✕ Supprimer"
     - Confirmer
     - ✅ Succès !
